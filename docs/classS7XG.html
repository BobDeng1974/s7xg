bool S7XG::macPower ( uint8_t  power)

Sets the transmission power.

According to datasheet: "it can be 2, 5, 8, 11, 14, 20 (non-915 band); 30, 28, 26, 24, 22, 20, 18, 16, 14, 12, 10 (915 band); 1 7, 16, 14, 12, 10, 7, 5, 2 (470 band)". Any value not supported by the current frequency band will result in an error. Note that some values might not be legal in your region.
